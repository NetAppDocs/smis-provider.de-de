= smis crsp
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Der `smis crsp` Befehl beschreibt von NetApp SMI-S Provider unterstützte CIM-registrierte Unterprofile, einschließlich NetApp SMI-S Provider-Unterprofile.

.Syntax
smis crsp

`[-t {*http* | *https*}]`

.Standort
`C:\Program Files (x86)\NetApp\smis\pegasus\bin`

.Berechtigungsebene
Ein Benutzer mit einem gültigen Benutzernamen und Kennwort

.Parameter
* `*[-t {http | https}]*`
+
Zu verwendenden Protokoll: HTTPS (Standard) oder HTTP



.Beispiel
